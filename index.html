Cadangan data (localStorage browser ini saja — tidak sinkron lintas perangkat)

Data tersimpan di browser ini saja. Kalau ganti device / hapus cache, datanya hilang. Export ke file JSON sesekali untuk backup; import untuk memulihkan.

Sinkron ke GitHub / Gist (opsional — pakai token pribadi)

Token disimpan HANYA di localStorage browser ini, dikirim langsung dari browser ke api.github.com (tidak lewat server manapun) — tapi tetap perlakukan seperti password. Pakai fine-grained token yang scope-nya dibatasi ke satu repo saja, jangan token dengan akses penuh semua repo.

Butuh scope repo (untuk simpan ke repo) dan/atau gist (untuk simpan ke Gist). Buat di github.com → Settings → Developer settings → Personal access tokens.
